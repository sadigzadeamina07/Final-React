## `✦` About the Project

> *"Elegance is not about being noticed, it's about being remembered."*

This project is a high-end, **Charlotte Tilbury**-inspired frontend for a luxury beauty web application. It features a responsive, aesthetic design optimised for an elegant shopping experience — crafted with the finest web technologies for optimal performance and visual refinement.

<br/>

## `⟡` Features

<br/>

| &nbsp; | Feature | Description |
|--------|---------|-------------|
| `✦` | **Premium UI / UX** | Sophisticated design with seamless sliders and fluid animations via Swiper |
| `⟡` | **Categorization** | Structured sections for makeup, skincare, and foundation products |
| `♡` | **Wishlist** | Interactive product saving via heart icons for a curated experience |
| `◇` | **Modern Architecture** | Built with industry-standard web technologies for speed and reliability |

<br/>

## `◇` Technologies

<br/>

```
  React v19   ·   Tailwind CSS   ·   Vite   ·   Swiper   ·   Axios
```

> *"Zəriflik göründüyün üçün yox, xatırlandığın üçündür."*
