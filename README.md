### <span style="color: #ff69b4;">🎀 About the Project</span>
This project is a high-end, **<span style="color: #c71585;">Charlotte Tilbury-inspired</span>** frontend for a beauty web application. It features a responsive, strictly aesthetic design optimized for an elegant shopping experience. It's giving absolute *luxury*, girlies! 💅✨💖

### <span style="color: #ff69b4;">🦢 Features</span>
- 🪞 **<span style="color: #db7093;">Premium UI/UX:</span>** Sophisticated design with seamless sliders and buttery smooth animations (Swiper).
- 💄 **<span style="color: #db7093;">Categorization:</span>** Elegantly structured sections for makeup, skincare, and foundation.
- 💌 **<span style="color: #db7093;">Wishlist Magic:</span>** Interactive product saving via aesthetic pink heart icons.
- ⚡ **<span style="color: #db7093;">Modern Architecture:</span>** Built with industry-standard web technologies to keep it fast and flawless.

### <span style="color: #ff69b4;">🩰 Technologies</span>
`React v19` ‧ `Tailwind CSS` ‧ `Vite` ‧ `Swiper` ‧ `Axios`

### <span style="color: #ff69b4;">💋 How to Run</span>

```bash
# 1. Clone the repository (Grab it girl!) 🍓
git clone <repo-url>

# 2. Install dependencies (Get ready...) 💄
npm install

# 3. Start development server (Slay!) 💅
npm run dev
```
