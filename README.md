### <span style="color: #c71585;">✦ About The Project</span>
This project is a high-end, **Charlotte Tilbury-inspired** frontend for a beauty web application. It features a responsive, strictly aesthetic design carefully engineered for an elegant and flawless shopping experience.

### <span style="color: #c71585;">✦ Key Features</span>
- 🪞 **<span style="color: #db7093;">Premium UI/UX:</span>** Sophisticated design tailored with seamless sliders and buttery smooth animations (Swiper).
- 💄 **<span style="color: #db7093;">Categorization:</span>** Elegantly structured, dynamic sections for makeup, skincare, and foundation.
- 💌 **<span style="color: #db7093;">Wishlist Integration:</span>** Interactive product saving functionality via aesthetic pink heart components.
- ⚡ **<span style="color: #db7093;">Modern Architecture:</span>** Built with industry-standard web technologies to ensure scalable and fast performance.

### <span style="color: #c71585;">✦ Installation</span>
```bash
git clone <repo-url> 
npm install          
npm run dev          
```

### <span style="color: #c71585;">✦ Layihə Haqqında</span>
Bu layihə **"Charlotte Tilbury"** konseptindən ilhamlanaraq yaradılmış, premium səviyyəli kosmetika e-ticarət platformasının frontend hissəsidir. Layihə tamamilə estetik dizayna və müasir istifadəçi təcrübəsinə əsaslanan peşəkar bir yanaşma ilə hazırlanmışdır.

### <span style="color: #c71585;">✦ Xüsusiyyətlər</span>
- 🪞 **<span style="color: #db7093;">Premium Dizayn:</span>** Yüksək keyfiyyətli istifadəçi interfeysi və axıcı (buttery smooth) animasiyalar.
- 💄 **<span style="color: #db7093;">Kateqoriyalar:</span>** Makiyaj və dəriyə qulluq məhsulları üçün xüsusi dizayn edilmiş zərif bölmələr.
- 💌 **<span style="color: #db7093;">İstəklər İntegrasiyası:</span>** Vizual cəhətdən xoş çəhrayı ürək butonları ilə məhsulların interaktiv olaraq seçilmişlərə əlavəsi.
- ⚡ **<span style="color: #db7093;">Müasir Memarlıq:</span>** Yüksək sürətli və etibarlı performans üçün sənaye standartlarına uyğun alətlərin istifadəsi.
