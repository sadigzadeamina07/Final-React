### <span style="color: #ff69b4;">🎀 About The Tea</span>
This is a high-end, **<span style="color: #c71585;">Charlotte Tilbury-inspired</span>** frontend web app. Strictly aesthetic design optimized for an elegant shopping experience. Giving absolute *luxury*, girlies! 💅✨💖

### <span style="color: #ff69b4;">🦢 Flawless Features</span>
- 🪞 **<span style="color: #db7093;">Premium UI:</span>** Seamless sliders & buttery smooth animations.
- 💄 **<span style="color: #db7093;">Categories:</span>** Elegantly structured makeup & skincare.
- 💌 **<span style="color: #db7093;">Wishlist Magic:</span>** Save products via aesthetic pink hearts.
- ⚡ **<span style="color: #db7093;">Tech Stack:</span>** `React v19`, `Tailwind`, `Vite`, `Swiper`, `Axios`. Fast & flawless.

